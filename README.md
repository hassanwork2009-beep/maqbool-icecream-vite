# Maqbool Ice Cream — Clean React + Vite

Static single-page React app. No SSR, no server, no Bun.

## Local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/`. Deployable as a static site on **Vercel**, **Netlify**, or **Cloudflare Pages** with zero configuration:

- **Build command:** `npm run build`
- **Output directory:** `dist`
